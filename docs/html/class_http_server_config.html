Detailed Description

Configuration settings for an HTTP server.

This class encapsulates configurable parameters for an HTTP server instance including network settings, logging, security features, performance tuning, and protocol limits. All settings are initialized with safe defaults.

The class supports the builder pattern for construction. Use HttpServerConfig::Builder to create configurations.
